### What is ERC Tokens
ERC (Ethereum Request for Comments) provides a common set of rules to ensure compatiblity with wallets, exchanges and smart contracts while facilitating the creation, issuance and management of tokens on the Ethereum blockchain
ERC20 tokens are the building blocks of the Ethereum ecosystem

### Common ERC Standars
- ERC-20: The standard for interchangeable (fungible) tokens
- ERC-721: The standard for non-fungible tokens (NFTs)
- ERC-1155: A multi-token standard that allows for both fungible and non-fungible tokens
- ERC-4626: A specialized tokenized vault standard for yield-bearing tokens
