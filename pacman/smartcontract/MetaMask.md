### What is MetaMask
MetaMask is **a popular non-custodial cryptocurrency wallet**, available as a browser extension and mobile app, designed to manage Ethereum and other blockchain assets (like ERC-20 tokens, NFTs, and Binance Smart Chain).
It **serves as a gateway** to Web3 ecosystem, providing access to decentralized applications (dApps), DeFi platforms, and NFT marketplaces

### Switch to a new MetaMask account when deploying to a new testnet
A new testnet provides you a new account pre-funded with free test ETH/Golier for development
1. Import an account
2. Reset localhost dapp connection by removing old one
3. Redeploy smartcontract to a new testnet if the current one use in-memory storage
