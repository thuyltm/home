### What is Smart Contract
A smart contract is simply a program that runs on the Ethereum blockchain. It's a collection of code (its functions) and data (its state) that resides at a specific address on the Ethereum blockchain

A smart contract is deployed to the network and run as programmed. **Application developers upload programs (reusable snippets of code) into EVM state, and users make requests to execute these code snippets with varying parameters.**

Smart contracts cannot be deleted by default, and interactions with them are irreversible

### Use Cases:
- Decentralized Finance (DeFi): Powering automated lending, borrowing, and trading
- Supply Chain: Releasing payments upon delivery verification
- Decentralized Application (dapp): An application built on a decentralized network that combines a smart contract and a frontend user interface
